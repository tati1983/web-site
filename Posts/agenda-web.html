Angular

Web App desarrollada en Angular. La aplicacion permite agregar tareas pendientes de un modo muy sencillo. Y eliminarlas una vez cumplido el objetivo


INTERFACE DE LA APP

Si no hay tareas agregadas vemos un mensaje que dice 'No hay tareas pendientes'

AGREGAR TAREAS

Los pendientes se tipean sobre el cuadro de texto y se agregan a la lista presionando 'Enter'

Las tareas se agregan en orden

ELIMINAR TAREAS

Una vez realizadas, podemos eliminarlas con el icono de la derecha
