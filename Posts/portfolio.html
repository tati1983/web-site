PORTFOLIO WEB

Angular / Java / MySQL

Esta es una aplicacion web, de arquitectura distribuida, diseñada y desarrollada como proyecto final del programa de formación desarrollado por el INTI en conjunto con la Camara Argentina del Software, 'Argentina Programa'. El FrontEnd fue desarrollado en Angular, integrando, ademas, algunas de las librerías mas populares, como Bootstrap y Material. El BackEnd fue programado en Java, con la ayuda de SpringBoot.


Ver el Codigo

FUNCIONES


  • Dar de alta un usuario: la aplicacion permite crear un usuario y generar una contraseña.
  • Cargar, modificar o eliminar datos del usuario: siendo la aplicacion un portfolio web, permite cargar datos del usuario referidos a su formacion, experiencia laboral, etc.
  • Imagenes: permite subir/bajar imagenes.
